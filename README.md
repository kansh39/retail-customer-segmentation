# 🛒 Retail Customer Segmentation & Analytics Dashboard

## 📌 Project Overview

This project analyzes retail transaction data to identify customer behavior patterns, segment customers based on value, and provide actionable business insights.

The project combines:
- RFM Analysis (Recency, Frequency, Monetary)
- K-Means Clustering
- Business Intelligence Visualization
- Streamlit Interactive Dashboard

---

## 🎯 Objective

To help businesses:
- identify high-value customers
- detect churn risk
- improve retention strategies
- personalize marketing campaigns

---

## 📊 Dataset

Online Retail Dataset containing:
- Customer purchases
- transaction history
- product-level data
- spending patterns

---

## ⚙️ Methodology

### 1️⃣ Data Cleaning
- removed cancelled orders
- handled missing customer IDs
- filtered negative quantity & price

### 2️⃣ Feature Engineering
- revenue calculation
- date transformation

### 3️⃣ RFM Analysis
- Recency
- Frequency
- Monetary segmentation

### 4️⃣ Customer Segmentation
- Champions
- Loyal Customers
- Lost Customers
- New Customers

### 5️⃣ K-Means Clustering
- unsupervised learning
- behavioral grouping
- cluster validation

### 6️⃣ Visualization
- segment distribution
- revenue contribution
- behavioral patterns
